Ceci est une maquette de l'application à fournir pour le projet "Application Parc".

    Remarques :
  • Tout appui sur les boutons "confirmation" ou "suppression" mènera à l'apparition d'une pop-up demandant la confirmation.
  • Pour commander un billet, le visiteur devra obligatoirement être connecté. Si ce n'est pas le cas, il sera amené à la page pour se connecter.
  • Dans ces exemples, le visiteur ne semble pas rester connecté. Il le sera avec la véritable application.
  • Le pointage n'apparait pas puisque le client dispose de bornes pour l'effectuer. Ces bornes sont néanmoins reliées à l'application pour permettre l'accès aux données.
